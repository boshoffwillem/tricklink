# tricklink
Video platform for uploading and viewing various tricks, stunts and talents by people.

## Tech stack
 - ASP.NET Core
 - ImageSharp: image processing
 - Xabe.FFmpeg: video processing
 - MediatR
 - Nuxt.js
 - Entity Framework Core
 - PostgreSQL
 - S3: file storage
 - Linode: cloud hosting
 - NGINX
